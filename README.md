# Simple NoFap App

This app is a simple NoFap that it's sole purpose is to enrouage users to not masturebate.
To do this the app will sent notifications to the User of the greater goal in life - which is to build a family.

---

# Main Features

The app will have the main features

## Time Counter

This will be a visual real time timer that displays the amount of seconds, minutes, hours, days, weeks, and months.

The more time the user accomalats, the fanciers and cooler the timer will look, to encourate user to keep going.

## Widget

The widget is there to not force the user to enter into the app to check the timer, and instead display the pretty Time Counter to the user, showing him his progress so far

## Encouraging notifications

Sometimes user might not realize how far he has come, maybe because he didn't bother to put in the widget.
In this case, we want to add constant notifications that encourage the user to keep going.

The user of course can configure the notifications, to not bother him.

The notifications for each milestore will be visually different as well in terms of sound effects
