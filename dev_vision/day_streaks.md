# Purpose

Day Streaks essentially have the following list of "purposes":
- To show user the milestone he has come, and thus encouraging him
- "Save NoFap streak" to allow the user to avoid resetting the timer

The main inspiration of the Day-Streaks is from Call of Duty's "Kill Streak" concept.
It rewards a player for managing to do something and gives me something cool to keep going with the "Kill Streak".
So we're essentially doing the same thing with Day Streaks, more or less.

Another way to view the 2nd purpose is following.
The point is not to force users to reset on every small mistake but encourage them to slowly stop succumbing to the urge, as a smoker needs to smoke less and less as time goes on until he stops completely.

# How they should look

Since the main inspiration is Kill Streaks from CoD, we want to present the Day Streaks as cool looking badges to honor the user for keeping up his no-fap.

We mainly need cool colourful icons for this.
